faq:
  sectionLabel: Support
  heading: Frequently asked
  description: >-
    Everything you need to know about getting started with the high-performance
    skeleton.
  items:
    - question: Why does semantic html impact seo?
      answer: >-
        Search engines use semantic tags (header, nav, article) to understand
        the context and hierarchy of your content. By using the correct tags,
        you make it easier for google to index your site accurately, leading to
        better rankings.
    - question: Does this work with my existing branding?
      answer: >-
        Yes. The black and white theme is a high-contrast base. You can easily
        inject your brand colors via the tailwind config in the head section to
        match your identity while maintaining accessibility standards.
    - question: Is there a cost for commercial use?
      answer: >-
        No. This skeleton is free to use for both personal and commercial
        projects. It is designed to be a starting point for any web project that
        values performance and inclusion.
    - question: Is this design blueprint mobile-responsive?
      answer: >-
        Yes, fully. We use tailwind's responsive prefixes (md:, lg:) to ensure
        the layout adapts gracefully to any screen size, from mobile phones to
        large desktop monitors.
    - question: Does it work with react or vue?
      answer: >-
        While this is a vanilla html/alpine.js design blueprint, the structure
        and tailwind classes can easily be copied into react components or vue
        templates. Alpine.js logic can be replaced with standard react/vue state
        management.
    - question: What about browser support?
      answer: >-
        This design blueprint relies on standard modern web features (flexbox,
        grid, es6). It works perfectly in all modern evergreen browsers (chrome,
        firefox, safari, edge).
